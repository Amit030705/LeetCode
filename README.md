# LeetCode Placement Platform 🚀

A full-stack web application built to help students prepare for coding interviews, placement drives, and DSA practice.

## ✨ Features

- 🔐 User Authentication & Login System
- 📚 Coding / Placement Preparation Resources
- 💻 LeetCode-style Problem Practice
- 📈 Progress Tracking
- ⚡ Fast and Responsive UI
- 📱 Mobile Friendly Design

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS / Tailwind CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
